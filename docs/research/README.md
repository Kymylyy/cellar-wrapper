# Research Archive

This directory stores archival research snapshots used during discovery and
design.

These files were part of the basis for the implementation, but they are not
the maintained source of truth for current public contracts. Some of them are
date-sensitive, some mix observed data with inference, and some predate the
later live-payload cleanup that informed the active docs.
